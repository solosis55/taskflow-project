
Experimentos de comparación con IA y sin IA

Problema 1- Aplicar Dark Mode en un proyecto//////////////

Sin IA:
Tareas realizadas:
-Crear el boton de cambio
-Diseñar el boton de cambio
-Añadir el cambio de clases al pulsar el boton y la persistencia
-Diseñar el Dark Mode en el CSS
-Resolución de bugs , errores y refactorización
-Añadir Comentarios

Tiempo total invertido : 20  min

Ventajas: 
-Comprensión completa de cada parte
-Aprendizaje práctico mas profundo
-Diseño mas humano

Desventajas:
-Alta probabilidad de bugs
-Codigo menos fiable
-Tiempo invertido alto
-Tiempo de busqueda y solución de errores alto

Con IA

Tareas realizadas:
-Utilizar Agent para crear el Dark Mode
-Verificar el código para detectar errorer

Ventajas:
-Tiempo invertido bajo
-Codigo robusto y consistente
-Probabilidad de bugs baja

Desventaja:
-Escaso aprendizaje práctico
-Posibles cambios no deseados
-Baja comprension del codigo generado

Tiempo ivertido: 5min aprox


Problema 2- Trasladar CSS a TailWind//////////////

Sin IA:
Tareas realizadas:
-Instalación y configuracion de TailWind
-Reconfiguración de clases en HTML
-Traspado de propiedades CSS a TailWind

Tiempo invertido: 33 min

Ventajas:
-Rediseño propio del HTML y del CSS
-Aprendizaje de frases TailWind

Desventajas:
-Tiempo invertido alto
-Bajo entendimiento de TailWind
-Posibilidad de bugs medio-alto

Con IA

Tareas realizadas:
-Utilizar Agent modificar el HTML y traspasar el CSS
-Verificar el código para detectar errores

Ventajas:
-Tiempo invertido bajo
-Probabilidad de bugs baja

Desventaja:
-Escaso aprendizaje práctico
-Alteración de ciertas animaciones/diseños del DOM
-Baja comprension del codigo TailWind

Tiempo ivertido: 8 min aprox


Problema 3- Creación de funciones reutilizables en JS/////////

Sin IA
Tareas realizadas:
-Documentación previa de las funciones necesarias y como crearlas
-Creación de funciones y puesta en práctica
