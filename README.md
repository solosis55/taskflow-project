\# TaskFlow



TaskFlow es una pequeña aplicación web de gestión de tareas desarrollada con \*\*HTML, CSS y JavaScript\*\*.

Permite crear, eliminar y organizar tareas directamente en el navegador con persistencia de datos mediante \*\*LocalStorage\*\*.



\## Demo



Puedes ver el proyecto funcionando aquí:



https://taskflow-project-five-cyan.vercel.app



\## Funcionalidades



\* Añadir nuevas tareas

\* Eliminar tareas

\* Marcar tareas como completadas

\* Cambiar prioridad de cada tarea

\* Menú interactivo de selección de prioridad

\* Persistencia de tareas usando LocalStorage

\* Filtro de búsqueda para encontrar tareas rápidamente

\* Interfaz responsive para dispositivos móviles



\## Tecnologías utilizadas



\* HTML5

\* CSS3

\* Flexbox

\* Media Queries

\* JavaScript (DOM Manipulation)

\* LocalStorage API



\## Cómo ejecutar el proyecto



1\. Clonar el repositorio



```bash

git clone https://github.com/tuusuario/taskflow.git

```



2\. Abrir la carpeta del proyecto



```bash

cd taskflow

```



3\. Abrir el archivo `index.html` en el navegador

&nbsp;  o usar \*\*Live Server\*\* en Visual Studio Code.



\## Estructura del proyecto



```

taskflow

│

├── index.html

├── styles.css

├── app.js

└── README.md

```



\* \*\*index.html\*\* → estructura de la aplicación

\* \*\*styles.css\*\* → estilos y diseño

\* \*\*app.js\*\* → lógica de interacción y manipulación del DOM



\## Características destacadas



\* Manipulación dinámica del DOM

\* Event delegation para manejar eventos en elementos dinámicos

\* Persistencia de datos con LocalStorage

\* Interfaz interactiva sin frameworks
